StashSync Settings

StashSync maps your Stash App Groups to Jellyfin Movies. Each Group becomes a Movie entry, with its scenes represented as chapters. After saving settings, run the "Sync Stash Groups" task from the Scheduled Tasks page.

Stash Connection

Base URL of your Stash instance. Do not include a trailing slash.
Found in Stash → Settings → Security → API Key.

TMDB Integration

Used to fetch poster and backdrop images directly from TheMovieDB during sync. Free to obtain at themoviedb.org/settings/api. If left blank, posters will fall back to the Stash cover image only.

Output

Directory where .strm files will be created. Add this path to Jellyfin as a Movies library. Jellyfin must have read/write access to this path.
When enabled, .strm files will contain Stash HTTP stream URLs. When disabled, local file paths from Stash will be used (requires shared filesystem).

StashProxy

URL of the StashProxy service. When set, .strm files will point to the proxy which streams all scenes in a Group as one continuous video using FFmpeg. Leave blank to use direct Stash stream URLs (plays first scene only). See the StashProxy README for setup instructions.

Filters

Groups with fewer scenes than this number will be skipped.
